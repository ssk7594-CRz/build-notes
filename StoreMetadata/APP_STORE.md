## Description

NextBuild는 여러 앱을 혼자 만들고 운영하는 사람을 위한 macOS 기능 메모 앱입니다.

앱별로 구현 예정 기능을 빠르게 추가하고, 완료된 기능은 접어둔 상태로 보관할 수 있습니다. 필요할 때 완료 영역을 펼치면 그동안 어떤 흐름으로 기능을 쌓아왔는지 확인할 수 있습니다.

주요 기능:
- 앱별 기능 카드
- 구현 예정 기능 빠른 추가
- 체크 한 번으로 구현 완료 이동
- 완료 기능 접기/펼치기
- 메뉴바와 플로팅 위젯
- 로컬 JSON 저장 및 백업

NextBuild는 계정 없이 로컬에서 동작합니다.
